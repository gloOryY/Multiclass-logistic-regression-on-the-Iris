# Multiclass-logistic-regression-on-the-Iris
Этот ноутбук демонстрирует мультиклассовую логистическую регрессию на классическом датасете Iris с использованием scikit-learn
Проект показывает полный путь классической задачи бинарной классификации:
по данным обследования пациента предсказать, есть ли риск сердечного заболевания (target = 1) или нет (target = 0), с помощью логистической регрессии.

Конвейер (pipeline):
Загрузка и первичный анализ данных.
Визуализация и исследование признаков.
Разделение на train/test.
Масштабирование признаков.
Обучение логистической регрессии.
Оценка качества модели: accuracy, confusion matrix, classification report, ROC/AUC.

Данные
Используется датасет heart.csv (классический heart disease dataset).
Каждая строка — один пациент.
Признаки (features), например:
age — возраст,
sex — пол,
trestbps — resting blood pressure,
chol — cholesterol,
thalach — максимальная достигнутая частота пульса, и др.

Целевая переменная:
target — 0 (нет заболевания), 1 (есть заболевание).
Файл heart.csv должен лежать в корне проекта (или путь должен совпадать с тем, что в коде).

Результаты
Модель логистической регрессии даёт:
Accuracy — доля правильных предсказаний на тестовой выборке.
Confusion matrix — показывает, сколько раз модель:
верно/неверно предсказала каждый класс.
Classification report — precision, recall, f1-score по каждому классу.
ROC-кривая и AUC — качество разделения классов по вероятностным оценкам модели.
